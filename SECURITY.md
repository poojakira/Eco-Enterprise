# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 8.2.x   | :white_check_mark: |
| 8.1.x   | :white_check_mark: |
| < 8.0.0 | :x:                |

## Reporting a Vulnerability

We take the security of the EcoTrack Nexus seriously. If you believe you have found a security vulnerability, please do NOT open a public issue. Instead, send a detailed report to `industrial-security@ecotrack-enterprise.dev`.

Our team will respond within 48 hours to coordinate a responsible disclosure and patch cycle.
